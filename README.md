A System and Method for Offline Emergency Alert Communication Using Device-to-Device Connectivity
In disaster situations such as earthquakes, floods, terrorist attacks, or network outages, conventional communication systems often fail due to infrastructure damage or network congestion. This project proposes a system that enables emergency alert communication without relying on centralized internet or cellular networks.

The system utilizes device-to-device (D2D) connectivity technologies such as Bluetooth, Wi-Fi Direct, or mesh networking to transmit emergency alerts between nearby devices. Messages are propagated across multiple devices in a decentralized manner, ensuring that critical information reaches affected individuals even in offline scenarios.
